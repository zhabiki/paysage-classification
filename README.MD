# Классификация пейзажей

Учебная задача классификация, в которой необходимо обучить модель разделять картинки пейзажей на 6 категорий — **здания, леса, горы, ледники, улицы, и моря**.

Датасет на Kaggle: https://www.kaggle.com/datasets/nitishabharathi/scene-classification

Оттуда мы используем только часть Train (~17 тысяч картинок), поскольку в Test отсутствуют метки. В качестве нейронной сети для дообучения используется **Resnet 50**.


## Установка

1. Скачайте и установите [Conda](https://docs.conda.io/projects/conda/en/stable/user-guide/install/index.html), если у вас её нет.
2. Создайте новое окружение Conda (`conda env create -f environment.yml`), после чего перейдите в него (`conda activate paysage-classification`).
3. Установите зависимости Python из файла: `pip install -r requirements.txt`
4. Вы молодец!!
